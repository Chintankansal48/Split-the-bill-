# Split the Bill Calculator

A free, simple tool to split a restaurant/group bill — either equally or by
custom share (how much each person actually ordered), including tip and tax.

## Live Demo
(Add your Vercel URL here after deploying)

## Features
- Split a bill equally across any number of people
- Custom split based on what each person ordered
- Add tip % and tax/GST % on top of the bill
- No sign-up, no backend, runs fully in the browser

## Tech
Plain HTML, CSS, and JavaScript — no build step, no dependencies.

## Run locally
Just open `index.html` in a browser.
